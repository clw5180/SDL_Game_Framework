### 1、说明
一个通过Camera类实现角色在地图行走的例子，也是一个支持Windows系统的简单游戏框架。


### 2、注意事项
- 链接器-输入-附加依赖项要包含SDL2相关的lib文件：
  - SDL2.lib
  - SDL2main.lib
  - SDL2_image.lib
  - SDL2_mixer.lib
  - SDL2_ttf.lib
- 最后要把SDL2相关的dll文件放在Windows系统目录或exe同一目录下！
